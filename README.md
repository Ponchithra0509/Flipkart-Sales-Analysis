# 📊 Flipkart Sales Analysis Dashboard

### 🚀 End-to-End Data Analyst Portfolio Project

**Tools Used:** SQL • Power BI • Power Query • DAX • GitHub

---

## 📌 Project Overview

This project analyzes Flipkart sales, products, customer ratings, pricing, and seller performance using SQL and Power BI.

The objective is to transform raw business data into interactive dashboards that provide meaningful business insights for decision-making.
